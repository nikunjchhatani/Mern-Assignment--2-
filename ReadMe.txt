=> You will need to install dependencies in Frontend as well as Backend folder using "npm install" in terminal of the respective folders (ex: "Mern Assignment (2) > Backend > npm install).

=> After installing dependencies you need to start both servers using npm start in both Backendand Frontend simultaneously in different terminals (ex: "Mern Assignment (2) > Backend > npm start).

=> After starting the server it should automatically redirect you to the website.
